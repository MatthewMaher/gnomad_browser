Downloads

Genome Aggregation Database (gnomAD) data are available at the links below. Please see this blog post for more information about this release.

Variant Dataset (VDS) format

Hail users can download gnomAD data quickly in Variant Dataset (VDS) format.

First, install Google Cloud SDK. Then use the Google Cloud Storage gsutil tool to copy partitions in parallel to your own storage bucket or local hard drive.

mkdir gnomad_data

To download individual exome sites VDS files (autosomes: 56.58 GB, chrX: 1.16 GB, chrY: 15.71 MB) 

gsutil -m cp -r gs://gnomad-public/release-170224/gnomad.exomes.r2.0.sites.release.autosomes.vds gnomad_data/
gsutil -m cp -r gs://gnomad-public/release-170224/gnomad.exomes.r2.0.sites.release.X.vds gnomad_data/
gsutil -m cp -r gs://gnomad-public/release-170224/gnomad.exomes.r2.0.sites.release.Y.vds gnomad_data/

To download individual genome sites VDS files (autosomes: 102.33 GB, chrX: 4.87 GB) 

gsutil -m cp -r gs://gnomad-public/release-170224/gnomad.genomes.r2.0.sites.release.autosomes.vds gnomad_data/
gsutil -m cp -r gs://gnomad-public/release-170224/gnomad.genomes.r2.0.sites.release.X.vds gnomad_data/

Or copy everything at once (164.96 GB) 

gsutil -m cp -r gs://gnomad-public/release-170224/* gnomad_data/

VCF and coverage files

To download VCFs and coverage the old-fashioned way as a 103 GB tar, click here or type on command line (this will take 12 hours or so...) 

curl -O https://data.broadinstitute.org/gnomAD/release-170227/gnomAD.release.170222.tar

To download specific chromosomes 

curl -O https://data.broadinstitute.org/gnomAD/release-170227/genomes/vcf/gnomad.genomes.r2.0.sites.1.vcf.bgz
